# GitQuest

ブラウザ上で Git コマンドを打ちながら、コミットグラフがリアルタイムで動く Git 学習アプリ。
ミッション形式でステップごとに Git を学べる。

## 機能
- 疑似ターミナル — ブラウザ上で git コマンドを入力できる
- コミットグラフ可視化 — コマンドに応じてブランチ・コミットがリアルタイムで描画される
- ミッションシステム — レベル別チュートリアルで段階的に学習できる
- 進捗管理 — どこまで完了したか記録・再開できる

## 技術スタック

| レイヤー | 技術 |
|---------|------|
| フロントエンド | Nuxt.js 3 + TypeScript |
| バックエンド | Spring Boot 3 (Java 21) |
| データベース | PostgreSQL 16 |
| 認証 | Spring Security + JWT |
| グラフ描画 | SVG (Canvas API) |

## ディレクトリ構成

```
gitquest/
├── frontend/    # Nuxt.js
├── backend/     # Spring Boot
└── docs/        # 設計資料・学習メモ
```
