# UI チェックコマンド

フロントエンドの実装が以下の方針を満たしているか確認する。

## チェック項目

- [ ] Tailwind CSS でスタイリングされているか
- [ ] モバイル（sm:）・タブレット（md:）・PC（lg:）のブレークポイントが設定されているか
- [ ] ダークテーマ（bg-gray-950 / bg-gray-900）ベースか
- [ ] ホバー・トランジションなどのインタラクションがあるか
- [ ] テキストが小さい画面でも読みやすいか

## 使い方

/ui-check

実行すると現在のフロントエンドコードを確認してチェック項目を評価する。
